## Текст теста

**1. Пользователь ищет «как сделать розу из фоамирана мастер-класс». Поисковик выдает видео, статьи с картинками и форумы, хотя слова «видео» или «картинки» в запросе отсутствовали. Какой компонент NLP отработал?**

а) Стемминг.

б) Классификация интента (понимание намерения пользователя получить визуальную инструкцию).

в) Удаление стоп-слов.

г) Латентно-семантическое индексирование.


**2. Какова основная задача стадии post-crawling у поискового бота перед добавлением документа в индекс?**

а) Расчет PageRank для найденных ссылок.

б) Проверка доступности сервера.

в) Лингвистическая нормализация: токенизация, лемматизация и очистка от HTML-тегов для создания чистого текстового потока.

г) Определение геолокации автора текста.


**3. В индексе находятся миллионы юридических документов. Почему классическое латентно-семантическое индексирование (LSI/SVD) может быть опасным при поиске конкретной статьи закона по номеру («статья 106 УК РФ»)?**

а) Оно слишком медленное для больших матриц.

б) Оно размывает точные идентификаторы, пытаясь найти «скрытый смысл» там, где нужна точная лексическая сверка (номер статьи — это не семантика, а ключ).

в) LSI не поддерживает кириллицу.

г) LSI требует GPU.


**4. При построении комплексной оценки релевантности модель объединяет Text Score = 0.9 и User Behavior Score = 0.1. Для конкретного пользователя документ на 10-й позиции идеален (он всегда покупает этот товар), но текстово он слаб. Что такое адаптивный персонализированный поиск в этом контексте?**

а) Увеличение веса Text Score до 1.0.

б) Игнорирование поведения пользователя ради объективности.

в) Динамическое изменение весов признаков ($w_{text}$ и $w_{behavior}$) на основе истории текущего сеанса поиска.

г) Переход исключительно на нейросетевое ранжирование.
