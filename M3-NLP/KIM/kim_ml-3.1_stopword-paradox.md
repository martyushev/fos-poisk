## Условия выполнения

**Формат работы:** Индивидуальная письменная работа.

**Регламент:** 10 минут. Объём ответа: 4–5 предложений.

**Инструкция для студента:** Напишите короткий аргументированный ответ инженеру, объясняющий, почему агрессивное удаление стоп-слов в медицинском поисковике может навредить точности поиска. Приведите конкретный пример термина, который пострадает.


## Текст задания

**«Парадокс стоп-слова»**

**Контекст:** Вы строите медицинский поисковик для врачей. Ваш инженер предлагает агрессивно удалить все стоп-слова для экономии места в инвертированном индексе.

**Задание:** Напишите короткий аргументированный ответ инженеру (4–5 предложений). Объясните, почему фраза «отказ левой почки без признаков диабета» после удаления стоп-слов превратится в проблему для точности поиска. Укажите конкретный пример термина, который пострадает (например, предлоги в названиях препаратов или анатомических связях).


## Критерии оценивания

| Критерий | Макс. балл | Описание |
| :--- | :--- | :--- |
| Аргументированность ответа | 3 | Объяснение содержит логическую цепочку: удаление стоп-слов → потеря смысловых связей → ошибки в поиске |
| Конкретный пример | 2 | Приведён конкретный термин или фраза, которая пострадает (предлог, союз, частица, меняющая смысл) |
| Четкость изложения | 1 | Ответ структурирован, лаконичен, соответствует ограничению по объёму |


## Эталон выполнения

**Пример правильного ответа:**

Удаление стоп-слов в медицинском поиске может катастрофически снизить точность, поскольку многие клинические фразы строятся на служебных частях речи. Фраза «отказ левой почки без признаков диабета» после удаления предлога «без» превратится в «отказ левой почки признаки диабета», что полностью меняет смысл — вместо отсутствия диабета поиск будет искать документы о диабете. Особенно критичны предлоги в названиях препаратов: запрос «препарат X при гипертонии» и «препарат X после гипертонии» — это разные клинические ситуации, которые стоп-фильтр сделает неразличимыми для системы.
