# Часть 1. Тестовые задания

## Вопрос 1

Основная цель персонализации поиска заключается в:

- а) уменьшении размера индекса;
- б) адаптации результатов поиска под конкретного пользователя;
- в) ускорении индексации документов;
- г) снижении нагрузки на сервер.

**Ответ:** б.

---

## Вопрос 2

Какие данные чаще всего используются для формирования пользовательского профиля?

- а) история запросов и переходов пользователя;
- б) характеристики процессора;
- в) скорость интернет-соединения;
- г) версия операционной системы.

**Ответ:** а.

---

## Вопрос 3

Что относится к мультимодальному поиску?

- а) поиск только по тексту;
- б) поиск по изображениям и голосовым запросам;
- в) поиск по SQL-запросам;
- г) поиск по IP-адресам.

**Ответ:** б.

---

## Вопрос 4

Какой метод наиболее часто применяется для поиска похожих изображений?

- а) BM25;
- б) TF-IDF;
- в) эмбеддинги, полученные нейронными сетями;
- г) PageRank.

**Ответ:** в.

---

## Вопрос 5

Корпоративный поиск предназначен для:

- а) поиска информации только в сети Интернет;
- б) поиска информации внутри корпоративных информационных ресурсов;
- в) поиска DNS-записей;
- г) поиска программных ошибок.

**Ответ:** б.

---

## Вопрос 6

Какой принцип является ключевым при персонализации поиска?

- а) хранение персональных данных без ограничений;
- б) соблюдение конфиденциальности персональных данных;
- в) увеличение объема индекса;
- г) сокращение времени индексации.

**Ответ:** б.

---

## Вопрос 7

Мультиязычный поиск позволяет:

- а) выполнять поиск сразу в нескольких базах данных;
- б) искать документы на различных языках;
- в) использовать несколько поисковых серверов;
- г) выполнять несколько запросов одновременно.

**Ответ:** б.

---

## Вопрос 8

Одним из недостатков персонализации является:

**Ответ:** б.
