## Рекомендации для преподавателя

1. **Модуль 5** является завершающим в курсе и предполагает синтез знаний из предыдущих модулей.
2. Лабораторная работа построена как развитие Лабораторной работы №3 (NLP) и №2 (ранжирование).
3. Рубежный кейс рекомендуется проводить в формате **питч-сессии** с приглашенными экспертами.
4. В деловых играх акцент делается не на код, а на **архитектурное мышление** и **экономическую обоснованность** решений.
