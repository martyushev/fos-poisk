## Условия выполнения

**Формат работы:** Индивидуальная письменная работа или электронное тестирование.

**Регламент:** 40 минут на выполнение всех 20 вопросов.

**Инструкция для студента:** Внимательно прочитайте каждый вопрос. Выберите один правильный вариант ответа из четырёх предложенных. Ответы занесите в бланк ответов.


## Текст теста

**1. Что такое персонализация поиска?**

а) Одинаковая выдача для всех пользователей.

б) Адаптация результатов поиска под индивидуальные предпочтения пользователя на основе его поведения и контекста.

в) Ранжирование документов только по текстовой релевантности.

г) Использование только демографических данных для выдачи результатов.


**2. Какие данные НЕ используются для построения профиля пользователя в персонализированном поиске?**

а) История поисковых запросов.

б) Клики по документам.

в) Личный пароль пользователя.

г) Время пребывания на странице (dwell time).


**3. Какая проблема возникает при чрезмерной персонализации поиска?**

а) Повышение релевантности выдачи.

б) Информационный пузырь (filter bubble) — пользователь видит только релевантный ему контент, что ограничивает разнообразие информации.

в) Увеличение скорости загрузки страницы.

г) Снижение стоимости разработки.


**4. Какой метод используется для обработки голосовых запросов?**

а) Только текстовый поиск.

б) Преобразование речи в текст (ASR) с последующей обработкой NLP.

в) Прямой поиск по аудиосигналу.

г) Использование только базы данных голосовых команд.


**5. Что такое CBIR (Content-Based Image Retrieval)?**

а) Поиск текста по изображению.

б) Поиск изображений на основе их визуального содержимого.

в) Поиск видео по звуковой дорожке.

г) Поиск документов по ключевым словам.


**6. В чём особенность поиска в корпоративных данных?**

а) Отсутствие необходимости в индексации.

б) Необходимость работы с разнородными источниками данных (PDF, электронные письма, чаты) и учётом прав доступа.

в) Использование только текстовых документов.

г) Отсутствие требований к безопасности.


**7. Какая этическая проблема связана с bias (смещением) в поисковых системах?**

а) Увеличение скорости поиска.

б) Воспроизведение и усиление существующих стереотипов и предубеждений в результатах выдачи.

в) Снижение затрат на разработку.

г) Улучшение качества изображений.


**8. Что такое информационная справедливость (information fairness) в поиске?**

а) Равное ранжирование всех документов независимо от их содержания.

б) Обеспечение равного доступа к информации для всех пользователей независимо от их характеристик.

в) Использование только общедоступных источников данных.

г) Отказ от персонализации.


**9. Как можно смягчить эффект информационного пузыря?**

а) Усилить персонализацию.

б) Внедрить механизмы разнообразия (diversity) в ранжирование и предлагать пользователю контент из разных категорий.

в) Отключить все алгоритмы ранжирования.

г) Использовать только глобальные признаки релевантности.


**10. Какой подход используется для обеспечения конфиденциальности данных пользователей в персонализированном поиске?**

а) Хранение всех данных в открытом виде.

б) Анонимизация данных, агрегация на уровне групп пользователей.

в) Передача данных третьим лицам.

г) Отказ от сбора любых данных.


**11. Какие модальности данных используются в мультимодальном поиске?**

а) Только текст.

б) Только изображения.

в) Текст, изображения, аудио, видео.

г) Только видео.


**12. В чём отличие персонализации от адаптации поиска?**

а) Это одно и то же.

б) Персонализация учитывает индивидуального пользователя, адаптация — контекстные факторы (устройство, геолокация, время).

в) Адаптация важнее персонализации.

г) Персонализация не использует данные о поведении пользователя.


**13. Какие методы используются для поведенческой сегментации пользователей?**

а) Кластеризация на основе истории взаимодействий.

б) Случайное разделение на группы.

в) Ручная классификация администратором.

г) Использование демографических данных.


**14. Какую роль играет геолокация в персонализированном поиске?**

а) Не влияет на выдачу.

б) Позволяет адаптировать результаты (например, показывать ближайшие магазины, локальный контент).

в) Используется только для налогообложения.

г) Заменяет все остальные признаки ранжирования.


**15. Как можно оценить качество персонализированной выдачи офлайн?**

а) Только через A/B-тест.

б) Используя метрики: персонализированный NDCG, разнообразие (diversity), новизна (novelty).

в) Только по CTR.

г) По времени загрузки страницы.


**16. Какие риски существуют при использовании генеративных моделей в персонализированном поиске?**

а) Отсутствие рисков.

б) Галлюцинации, потеря намерения пользователя, увеличение latency.

в) Только
