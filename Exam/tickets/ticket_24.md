# Экзаменационный билет № 24

## Часть 1. Теоретический вопрос (30 баллов)

**Что такое прозрачность и объяснимость ранжирования в поисковых системах? Как они достигаются?**

*Рекомендация по ответу:*

- **Прозрачность:** способность объяснить, почему документ оказался на конкретной позиции
- **Методы:** SHAP, LIME для feature attribution
- **Визуализация:** веса внимания в трансформерах
- **Цель:** доверие пользователей, аудит, выявление bias
- **Ограничения:** сложность интерпретации глубоких моделей

## Часть 2. Практическая задача (40 баллов)

**Анализ explainability**

**Сценарий:** пользователь спрашивает, почему товар не в топе

**Ответ:**
1. **Важность признаков:** высокий вклад цены (товар дорогой)
2. **SHAP-анализ:** значение признаков для данного документа
3. **Рекомендация:** снижение цены или улучшение текстового описания


## Часть 3. Дополнительный вопрос (30 баллов)

**Как SHAP и LIME помогают в объяснимости ранжирования?**

- **SHAP:** значения Shapley для каждого признака — вклад в предсказание
- **LIME:** локальные аппроксимации — объяснение для конкретного документа
- **Сравнение:** SHAP — теоретически обоснован, LIME — быстрее
