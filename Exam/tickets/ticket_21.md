# Экзаменационный билет № 21

## Часть 1. Теоретический вопрос (30 баллов)

**Как решается проблема холодного старта в персонализированных поисковых системах?**

*Рекомендация по ответу:*

- **Холодный старт:** отсутствие данных о новом пользователе или товаре
- **Для нового пользователя:** контентные признаки вместо поведенческих, коллективная фильтрация от похожих пользователей
- **Для нового товара:** использование контентных эмбеддингов (текст + изображение), сходство с популярными товарами
- **Стратегии:** explore-exploit, быстрое накопление данных


## Часть 2. Практическая задача (40 баллов)

**Решение проблемы холодного старта**

**Сценарий:** новый товар в интернет-магазине

**Решение:**
1. **Контентные признаки:** эмбеддинг описания, категория, цена
2. **Похожие товары:** поиск по эмбеддингу → ранжирование по сходству
3. **Explore-exploit:** показ новым товарам с небольшим бонусом в ранжировании
4. **Быстрый сбор данных:** A/B-тест для новых товаров


## Часть 3. Дополнительный вопрос (30 баллов)

**Что такое Content-based filtering и как он помогает при холодном старте?**

- **Content-based filtering:** рекомендации на основе характеристик товара
- **Преимущество:** не требует истории взаимодействий
- **Пример:** новый фильм → рекомендация по жанру, актёрам
