# Экзаменационный билет № 7

## Часть 1. Теоретический вопрос (30 баллов)

**Что такое BERT? Опишите архитектуру, этапы предобучения и применение в информационном поиске.**

*Рекомендация по ответу:*

- **Архитектура:** Transformer Encoder (12/24 слоя), bidirectional контекст
- **MLM (Masked Language Model):** маскирование 15% токенов, предсказание по контексту
- **NSP (Next Sentence Prediction):** предсказание следующего предложения
- **CLS-токен:** агрегированное представление для классификации
- **Применение в поиске:** эмбеддинги документов, ранжирование, понимание запроса


## Часть 2. Практическая задача (40 баллов)

**Fine-tuning BERT (псевдокод)**

Опишите алгоритм fine-tuning BERT для intent classification.

**Этапы:**
1. Загрузка предобученной модели (BertForSequenceClassification)
2. Токенизация запросов (padding, truncation)
3. Настройка Trainer (learning rate, batch size, epochs)
4. Обучение с валидацией
5. Оценка (accuracy, F1)


## Часть 3. Дополнительный вопрос (30 баллов)

**Что такое CLS-токен и как он используется?**

- **CLS-токен:** специальный токен в начале последовательности
- **Назначение:** агрегирует информацию из всего предложения через Self-Attention
- **Использование:** подаётся в классификационную голову для задач классификации
