## Часть 1. Теоретический вопрос (30 баллов)

**Опишите классическую архитектуру поисковой системы. Какие компоненты входят в её состав и как они взаимодействуют?**

*Рекомендация по ответу:*

- **Краулер:** обход страниц, politeness policy, deduplication
- **Индексатор:** токенизация, построение инвертированного индекса
- **Ранкер:** этапы retrieval → reranking, модели (TF-IDF, BM25, ML)
- **Сервер выдачи:** обработка запросов, сниппеты, кэширование

*Схема:* Краулер → Индексатор → Индекс → Ранкер ← Сервер выдачи ← Пользователь


## Часть 2. Практическая задача (40 баллов)

**Расчёт NDCG@5**

Релевантность: [1, 0, 1, 1, 0]

**a) Precision@3 и Recall@3:**
- Precision@3 = 2/3 ≈ 0.67
- Recall@3 = 2/3 ≈ 0.67

**б) NDCG@5:**
- DCG@5 = 1 + 0 + 0.5 + 0.4307 = 1.9307
- IDCG@5 = 1 + 0.6309 + 0.5 = 2.1309
- NDCG@5 = 1.9307 / 2.1309 ≈ 0.906
