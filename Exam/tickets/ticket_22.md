# Экзаменационный билет № 22

## Часть 1. Теоретический вопрос (30 баллов)

**Опишите гибридный поиск (dense + sparse). В чём преимущества комбинирования подходов?**

*Рекомендация по ответу:*

- **Sparse поиск:** BM25, TF-IDF — точное совпадение терминов, интерпретируемость
- **Dense поиск:** эмбеддинги (SBERT, DPR) — учёт семантики, синонимов
- **Гибридный подход:** комбинация скорингов Sparse и Dense
- **Преимущества:** точность BM25 + семантическая гибкость эмбеддингов
- **Реализация:** линейная комбинация или обучение ранкеру

## Часть 2. Практическая задача (40 баллов)

**Проектирование гибридной системы**

**Компоненты:**
1. **Sparse:** BM25 + инвертированный индекс
2. **Dense:** эмбеддинги документов + FAISS-индекс
3. **Гибридный скоринг:** Score = α·BM25_score + β·Dense_score
4. **Ранжирование:** объединённый список → Cross-Encoder для reranking


## Часть 3. Дополнительный вопрос (30 баллов)

**Что такое Query expansion и для чего он используется?**

- **Query expansion:** расширение запроса синонимами и связанными терминами
- **Цель:** повышение полноты поиска
- **Методы:** WordNet, doc2query, LLM-генерация
