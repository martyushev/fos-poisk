# Экзаменационный билет № 12

## Часть 1. Теоретический вопрос (30 баллов)

**Как генеративные модели (T5, BART, GPT) используются в поиске для query rewriting? Какие риски существуют?**

*Рекомендация по ответу:*

- **Query rewriting:** переформулирование запросов, генерация альтернатив
- **Применение:** расширение синонимами, исправление формулировок
- **Риски:** галлюцинации (несуществующие термины), потеря намерения, увеличение latency
- **Валидация:** фильтрация по словарю, классификатор интента


## Часть 2. Практическая задача (40 баллов)

**Query rewriting**

Запрос: «почему у меня не работает принтер L3250 после обновления Windows 11»

**Варианты переформулирования:**
1. «принтер Epson L3250 не работает обновление Windows 11»
2. «ошибка принтера L3250 после win11 update»

**Риски:** подмена модели принтера, потеря слова «не работает»

**Смягчение:** проверка сохранения ключевых сущностей (NER)
