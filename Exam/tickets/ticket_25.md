# Экзаменационный билет № 25

## Часть 1. Теоретический вопрос (30 баллов)

**Как оценивается качество персонализированной выдачи офлайн и онлайн? Какие метрики наиболее релевантны?**

*Рекомендация по ответу:*

- **Офлайн-метрики:** персонализированный NDCG, diversity@k, novelty@k
- **Онлайн-метрики:** CTR, конверсия, retention, время на сайте
- **A/B-тест:** контрольная группа (без персонализации) и экспериментальная
- **Особенности:** учёт информационного пузыря, баланс релевантности и разнообразия

## Часть 2. Практическая задача (40 баллов)

**Сценарий A/B-теста для персонализации**

**Цель:** оценить влияние персонализации на поведение пользователей

**Дизайн:**
1. **Контрольная группа:** без персонализации
2. **Экспериментальная группа:** с персонализацией
3. **Метрики:** CTR, время на сайте, конверсия
4. **Длительность:** 2 недели
5. **Анализ:** t-test, доверительные интервалы


## Часть 3. Дополнительный вопрос (30 баллов)

**Какие риски существуют при A/B-тестировании персонализации?**

- **Interference:** влияние между группами
- **Сезонность:** разное поведение в разные дни
- **Новизна:** краткосрочный эффект новизны
- **Решение:** изоляция групп, учёт сезонности, extended A/B-тест
