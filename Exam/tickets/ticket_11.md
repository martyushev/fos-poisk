# Экзаменационный билет № 11

## Часть 1. Теоретический вопрос (30 баллов)

**Опишите пайплайн понимания запроса (Query Understanding). Какие задачи в него входят?**

*Рекомендация по ответу:*

- **Spell correction:** исправление опечаток (символьные n-граммы, T5)
- **NER (Named Entity Recognition):** извлечение сущностей (BERT, BIO-разметка)
- **Intent classification:** классификация намерений (fine-tuned BERT, 5–7 классов)
- **Query expansion:** генерация синонимов (doc2query, BERT masked LM)
- **Интеграция:** последовательная или параллельная обработка


## Часть 2. Практическая задача (40 баллов)

**Проектирование пайплайна Query Understanding**

Для онлайн-кинотеатра спроектируйте пайплайн понимания запроса.

**Этапы:**
1. **Spell correction:** исправление опечаток в названиях фильмов
2. **NER:** выделение жанров, актёров, режиссёров
3. **Intent classification:** поиск, рекомендация, информация
4. **Query expansion:** синонимы (комедия → юмор, смешное)


## Часть 3. Дополнительный вопрос (30 баллов)

**Как Intent Classification изменяет ранжирование документов?**

- Разные интенты → разные приоритеты признаков
- Поиск: текстовое сходство
- Рекомендация: поведенческие сигналы
- Навигация: точное совпадение
