# Экзаменационный билет № 15

## Часть 1. Теоретический вопрос (30 баллов)

**Объясните методы кластеризации текстов (KMeans). Как выбирается оптимальное число кластеров? Для чего используется кластеризация в поисковых системах?**

*Рекомендация по ответу:*

- **KMeans:** итерационный алгоритм, минимизация внутрикластерной дисперсии
- **Выбор k:** метод локтя (inertia), коэффициент силуэта (silhouette score)
- **Интерпретация:** топ-10 слов по центру кластера (cluster_centers_)
- **Применение:** группировка документов по темам, категоризация, анализ тем
- **Визуализация:** t-SNE/UMAP для 2D-проекции


## Часть 2. Практическая задача (40 баллов)

**Анализ кластеров**

Для 5 кластеров получены топ-10 ключевых слов. Проанализируйте темы.

**Пример кластеров:**
1. «футбол гол чемпионат матч» → Спорт
2. «акции рынок биржа индекс» → Финансы
3. «процессор память диск ноутбук» → Технологии
4. «фильм актёр режиссёр кино» → Кино
5. «рецепт готовить ингредиенты» → Кулинария


## Часть 3. Дополнительный вопрос (30 баллов)

**Как wordcloud помогает интерпретировать результаты кластеризации?**

- Визуализирует частоту ключевых слов (размер = важность)
