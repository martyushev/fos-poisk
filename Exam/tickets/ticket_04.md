# Экзаменационный билет № 4

## Часть 1. Теоретический вопрос (30 баллов)

**Объясните различия между pointwise, pairwise и listwise подходами к обучению ранжированию. Приведите примеры функций потерь.**

*Рекомендация по ответу:*

- **Pointwise:** регрессия/классификация для каждого документа; MSE, LogLoss
- **Pairwise:** попарное сравнение документов; RankNet (кросс-энтропия), RankSVM
- **Listwise:** оптимизация метрики списка; LambdaRank, ListNet, ListMLE
- **Сравнение:** качество (listwise ≥ pairwise ≥ pointwise), сложность (обратная зависимость)


## Часть 2. Практическая задача (40 баллов)

**Анализ дисбаланса релевантности**

Распределение: 85% — нерелевантные, 10% — частично релевантные, 5% — высокорелевантные.

**Методы коррекции:**
- **Upsampling:** увеличение числа примеров редких классов
- **Взвешивание:** увеличение веса ошибки для редких классов
- **Специализированные метрики:** NDCG вместо Accuracy


## Часть 3. Дополнительный вопрос (30 баллов)

**Какие метрики качества используются для оценки ранжирования?**

- **Precision:** доля релевантных среди выданных
- **Recall:** доля найденных среди всех релевантных
- **MAP:** средняя точность с учётом позиции
- **MRR:** обратный ранг первого релевантного
- **NDCG:** взвешенная по позиции релевантность
