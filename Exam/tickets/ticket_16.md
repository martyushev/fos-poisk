# Экзаменационный билет № 16

## Часть 1. Теоретический вопрос (30 баллов)

**Какие этические проблемы существуют в информационном поиске? Объясните понятия bias, информационная справедливость, конфиденциальность данных.**

*Рекомендация по ответу:*

- **Bias:** систематическое смещение результатов (пол, раса, география), возникающее из-за нерепрезентативных данных
- **Информационная справедливость:** равный доступ к информации для всех пользователей независимо от характеристик
- **Конфиденциальность:** защита данных пользователей (анонимизация, GDPR, дифференциальная приватность)
- **Прозрачность:** объяснимость ранжирования (SHAP, LIME, feature attribution)
- **Информационный пузырь:** ограничение разнообразия информации


## Часть 2. Практическая задача (40 баллов)

**Анализ bias**

Поисковая система показывает для запроса «CEO» в основном мужчин.

**Анализ проблемы:**
1. **Причина:** датасет обучения содержит гендерное смещение
2. **Пострадавшие:** женщины, ищущие информацию о CEO
3. **Измерение:** demographic parity, equal opportunity
4. **Исправление:** сбалансированный датасет, fairness-регуляризация


## Часть 3. Дополнительный вопрос (30 баллов)

**Что такое информационный пузырь и как его смягчить в персонализированном поиске?**

- **Информационный пузырь:** пользователь видит только релевантный ему контент
- **Смягчение:** механизмы diversity, случайные экспозиции, баланс релевантности и новизны
