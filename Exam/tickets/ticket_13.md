# Экзаменационный билет № 13

## Часть 1. Теоретический вопрос (30 баллов)

**Какие этапы предобработки текста применяются в поисковых системах? Объясните токенизацию, лемматизацию, стемминг и удаление стоп-слов.**

*Рекомендация по ответу:*

- **Токенизация:** разбиение текста на токены (слова, пунктуация)
- **Нормализация:** приведение к нижнему регистру, удаление HTML-тегов
- **Стоп-слова:** удаление общеупотребительных слов (кроме медицинских/юридических доменов)
- **Стемминг:** обрезание окончаний (Porter, Snowball) — быстрее, но грубее
- **Лемматизация:** приведение к начальной форме (pymorphy2, WordNet) — медленнее, но точнее


## Часть 2. Практическая задача (40 баллов)

**Кластеризация документов**

Для 5000 текстовых документов опишите пайплайн кластеризации.

**Этапы:**
1. **Предобработка:** токенизация, лемматизация, стоп-слова
2. **Векторизация:** TF-IDF (max_df=0.9, min_df=2, ngram_range=(1,2))
3. **Снижение размерности:** TruncatedSVD → 100 компонент
4. **Кластеризация:** KMeans, выбор k (локоть + силуэт)
5. **Интерпретация:** топ-10 слов по кластерам


## Часть 3. Дополнительный вопрос (30 баллов)

**В каких доменах удаление стоп-слов может быть вредно для поиска?**

- **Медицина:** «без» в «отказ почки без признаков диабета» меняет смысл
- **Юриспруденция:** предлоги в «право собственности на имущество»
- **Химия:** предлоги в названиях соединений
