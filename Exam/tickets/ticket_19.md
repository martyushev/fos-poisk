# Экзаменационный билет № 19

## Часть 1. Теоретический вопрос (30 баллов)

**В чём особенности корпоративного поиска? Какие источники данных и форматы документов характерны для него?**

*Рекомендация по ответу:*

- **Особенности:** разнородные источники, права доступа, безопасность, конфиденциальность
- **Источники:** Confluence, Sharepoint, электронные письма, чаты, файловые системы
- **Форматы:** HTML, PDF, Word, Excel, JSON, сообщения чатов
- **Проблемы:** извлечение чистого текста из вложений, иерархия структуры
- **Сложности:** учёт прав доступа в ранжировании, обновление индекса


## Часть 2. Практическая задача (40 баллов)

**Проектирование корпоративного поиска**

**Компоненты:**
1. **Краулер:** обход Confluence, файловых систем, почты
2. **Индексатор:** извлечение текста (PDF, Word, HTML), построение индекса
3. **Ранжирование:** текстовое сходство + авторитетность (частота доступа)
4. **Безопасность:** фильтрация по правам доступа на этапе поиска
5. **Обновление:** инкрементальная индексация


## Часть 3. Дополнительный вопрос (30 баллов)

**Как учитываются права доступа в корпоративном поиске?**

- **На этапе индексации:** сохранение ACL (Access Control List)
- **На этапе поиска:** фильтрация результатов по правам пользователя
- **Проблема:** увеличение latency
- **Решение:** кэширование прав, предварительная фильтрация
