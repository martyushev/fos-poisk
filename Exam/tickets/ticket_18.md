# Экзаменационный билет № 18

## Часть 1. Теоретический вопрос (30 баллов)

**Что такое мультимодальный поиск? Опишите подходы к поиску по изображениям (CBIR, deep visual embeddings) и голосовому поиску (ASR).**

*Рекомендация по ответу:*

- **Мультимодальный поиск:** поиск по данным разных модальностей (текст, изображения, аудио, видео)
- **CBIR (Content-Based Image Retrieval):** поиск по визуальному содержимому (цвет, текстура, форма)
- **Deep visual embeddings:** CNN для извлечения признаков изображений (ResNet, ViT)
- **Голосовой поиск:** ASR (Speech-to-Text) + NLP
- **Мультимодальные модели:** CLIP, ViT (объединение текста и изображений)


## Часть 2. Практическая задача (40 баллов)

**Анализ этических проблем мультимодального поиска**

**Проблемы:**
1. **Bias в изображениях:** стереотипные результаты поиска
2. **Конфиденциальность:** распознавание лиц без согласия
3. **Культурная敏感性:** разное восприятие изображений

**Методы смягчения:**
- Аудит датасетов
- Анонимизация лиц
- Локализация (учёт культурных особенностей)


## Часть 3. Дополнительный вопрос (30 баллов)

**Как работает CBIR? Что такое deep visual embeddings?**

- **CBIR:** поиск по визуальным характеристикам (цвет, текстура, форма)
- **Deep visual embeddings:** векторы, извлечённые CNN (ResNet, ViT)
- **Процесс:** изображение → CNN → эмбеддинг → поиск ближайших соседей
