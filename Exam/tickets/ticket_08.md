# Экзаменационный билет № 8

## Часть 1. Теоретический вопрос (30 баллов)

**Объясните принципы векторного поиска. Какие алгоритмы ANN (HNSW, IVF) используются и как они работают?**

*Рекомендация по ответу:*

- **Векторный поиск:** поиск ближайших соседей в многомерном пространстве эмбеддингов
- **Проблема:** полный перебор O(N×d) для миллионов документов
- **HNSW (Hierarchical Navigable Small World):** многослойная графовая структура, жадный поиск
- **IVF (Inverted File Index):** кластеризация (k-means), поиск в ближайших кластерах
- **Библиотеки:** FAISS, ScaNN
