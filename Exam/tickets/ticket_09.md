# Экзаменационный билет № 9

## Часть 1. Теоретический вопрос (30 баллов)

**Какие метрики используются для оценки качества ранжирования? Объясните формулы Precision, Recall, MAP, MRR, NDCG.**

*Рекомендация по ответу:*

- **Precision:** TP / (TP + FP) — доля релевантных среди выданных
- **Recall:** TP / (TP + FN) — доля найденных среди всех релевантных
- **MAP:** средняя точность по всем запросам
- **MRR:** 1 / rank первого релевантного
- **NDCG:** DCG / IDCG, где DCG = Σ rel_i / log₂(i+1)


## Часть 2. Практическая задача (40 баллов)

**Расчёт MAP**

Релевантность: [1, 0, 1, 1, 0]

**Точность на позициях (P@k):**
- P@1 = 1/1 = 1.0
- P@3 = 2/3 ≈ 0.667
- P@4 = 3/4 = 0.75

**Average Precision (AP):**
- AP = (1.0 + 0.667 + 0.75) / 3 ≈ 0.806

**MAP = AP (для одного запроса)**


## Часть 3. Дополнительный вопрос (30 баллов)

**В чём отличие F-меры от E-меры?**

- **F-мера:** среднее гармоническое Precision и Recall (F1 = 2PR/(P+R))
- **E-мера:** взвешенная мера эффективности, учитывает приоритет Precision или Recall (E = 1 - (1/αP + (1-α)/R))
