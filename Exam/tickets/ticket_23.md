# Экзаменационный билет № 23

## Часть 1. Теоретический вопрос (30 баллов)

**Что такое domain adaptation и как она применяется для поисковых моделей на вертикальных доменах (медицина, юриспруденция, e-commerce)?**

*Рекомендация по ответу:*

- **Domain adaptation:** перенос знаний из общего домена в специализированный (медицина, право)
- **Методы:** fine-tuning предобученных моделей (BERT, SBERT) на данных целевого домена
- **Особенности:** специализированная терминология, точность выше, чем у общих моделей
- **Сложности:** дефицит размеченных данных, регуляторные требования

## Часть 2. Практическая задача (40 баллов)

**Проектирование вертикального поиска**

**Задача:** поиск по юридическим документам

**Архитектура:**
1. **Fine-tuning:** BERT на юридических текстах (статьи законов, судебные решения)
2. **Индексация:** эмбеддинги документов → FAISS
3. **Специализированные признаки:** NER (статьи, номера дел), BM25
4. **Ранжирование:** гибридный поиск + Cross-Encoder


## Часть 3. Дополнительный вопрос (30 баллов)

**В чём отличие fine-tuning от обучения с нуля?**

- **Fine-tuning:** дообучение предобученной модели на целевом домене
- **Преимущества:** требуется меньше данных, быстрее, лучше обобщает
- **Обучение с нуля:** требуется большой объём данных и вычислительных ресурсов
