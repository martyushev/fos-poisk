**Занятие:** 2.2. Обучение ранжированию / 2.4. Оценка качества ранжирования

## Тема

Реализация алгоритма ранжирования на основе машинного обучения

---

## Цель работы

Реализовать полный пайплайн Learning to Rank (LTR): от предобработки сырых данных до оптимизации метрики качества списка (NDCG) с помощью градиентного бустинга. Научиться управлять процессом ранжирования через инженерию признаков и гиперпараметры.

---

## Инструментарий

- Python
- Jupyter Notebook
- pandas, numpy
- scikit-learn
- lightgbm, xgboost

---

## Задания

1. Загрузить логи поиска (query, document, clicks, position), сгруппировать документы по query_id.
2. Сгенерировать матрицу признаков:
   - BM25 сходство запроса и заголовка;
   - поведенческий признак: CTR × 1/log(1 + position);
   - статический признак: длина документа.
3. Задать целевую переменную релевантности (y) на основе кликов (клик = 2, показ без клика = 1).
4. Обучить модель HistGradientBoostingRegressor в pointwise-режиме, отсортировать группы документов и зафиксировать NDCG@5 как baseline.
5. Переобучить модель с использованием LGBMRanker (objective='lambdarank'), оптимизируя ndcg при eval_at=[1, 5, 10].
6. Подобрать гиперпараметры (глубина деревьев, количество листьев) для максимизации NDCG@5 и сравнить с baseline.
