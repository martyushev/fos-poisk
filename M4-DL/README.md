## Рекомендации для преподавателя

- Модуль 4 является ключевым в курсе и предполагает глубокое погружение в нейросетевые архитектуры.
- Лабораторные работы выполняются последовательно, каждая следующая опирается на результаты предыдущей.
- Контрольная работа включает расчёт метрик IR (NDCG, MRR, Precision, Recall) и проектирование архитектур.
- На экзамене акцент делается на устное обоснование выбора архитектур и анализ trade-off качество/latency.
- Для оценивания лабораторных работ используется рубрикатор, приведённый в ФОС.
