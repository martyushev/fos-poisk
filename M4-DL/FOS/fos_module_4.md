# Модуль 4. Нейросетевые модели в поиске и ранжировании (IR, QA)

**Профессиональная роль:** ML Researcher / Search Engineer

---

## Назначение ФОС

Фонд оценочных средств предназначен для оценки уровня сформированности профессиональных компетенций обучающихся в области применения нейросетевых трансформерных архитектур для задач информационного поиска (IR) и вопросно-ответных систем (QA).

В рамках модуля оценивается способность обучающегося:

- выбирать архитектуры (Bi-Encoder, Cross-Encoder) для решения задач поиска и ранжирования;
- проектировать конвейеры понимания запросов (query understanding pipeline);
- проводить тонкую настройку (fine-tuning) моделей и оценивать их качество с использованием метрик IR;
- анализировать компромиссы между качеством и производительностью;
- выявлять и интерпретировать ошибки нейросетевых моделей.

## Методические рекомендации

1. Все оценочные средства должны проводиться в соответствии с календарным планом модуля.
2. Лабораторные работы выполняются как в малых группах (2–3 человека), так и индивидуально (по усмотрению преподавателя).
3. Контрольная работа проводится в письменной форме; разрешено использование калькуляторов и справочных материалов.
4. Экзамен проводится в устной форме с решением практической задачи у доски.
5. Для оценивания используется рубрикатор, приведённый выше.
6. При оценке лабораторных работ особое внимание обращается не только на финальные метрики, но и на понимание студентом ограничений своего решения.

- Python 3.8+.
- Библиотеки: scikit-learn, pandas, numpy, nltk, sentence-transformers, faiss, torch, transformers.
