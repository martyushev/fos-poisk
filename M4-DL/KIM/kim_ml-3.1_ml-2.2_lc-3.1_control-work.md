## Текст теста

### Задача 1. (2 балла) Архитектуры

Даны две поисковые системы:

- Система А использует Bi-Encoder на базе BERT.
- Система В использует Cross-Encoder на базе BERT.

Ответьте на вопросы:

a) Какая система обеспечит меньший latency при обработке 10 000 запросов в секунду к коллекции из 10 млн документов? Обоснуйте.

b) Какая система даст более высокое качество ранжирования (NDCG)? Обоснуйте.

c) Предложите гибридную архитектуру, сочетающую преимущества обеих.


### Задача 2. (3 балла) Понимание запроса

Запрос пользователя: «яблоко падает»

a) Приведите минимум 3 различных намерения, которые может иметь пользователь.

b) Как BERT помогает разрешить полисемию в этом запросе? Опишите механизм на уровне attention-голов.

c) Предложите способ явного извлечения намерения (классификация) и покажите, как это изменит ранжирование.


### Задача 3. (2 балла) Метрики

Даны 5 документов, ранжированных системой. Релевантность (по экспертной оценке): [1, 0, 1, 1, 0]. Вычислите:

a) Precision@3 и Recall@3;

b) NDCG@5 (с использованием логарифмического дисконта).


### Задача 4. (3 балла) Генеративное переформулирование

Исходный запрос: «почему у меня не работает принтер ервоп l3250 после обновления win11»

a) Предложите 2 варианта переформулирования через генеративную модель (T5/BART).

b) Какие риски могут возникнуть при таком переформулировании?

c) Предложите способ валидации качества переформулирования (автоматическая метрика и/или human-in-the-loop).


## Ключи для преподавателя

**Задача 1:**

- a) **Bi-Encoder:** эмбеддинги документов пре-вычисляются, поиск — ANN в FAISS, latency O(log N). Cross-Encoder требует прогона через сеть для каждой пары, latency O(N · L).

- б) **Cross-Encoder:** видит совместный контекст (query + doc), качество выше на 5–15% NDCG.

- в) **Двухэтапная схема:** Bi-Encoder для candidate generation (top-100), Cross-Encoder для reranking.

**Задача 2:**

- a) 1) Физика (закон всемирного тяготения); 2) Сорт яблока; 3) Песня/фильм «Яблоко падает».

- б) BERT через механизм self-attention определяет связи между словами и их контекст; каждая голова attention фиксирует разные аспекты зависимости между словами.

- в) Fine-tuned BERT для классификации интента. Ранжирование изменяется: при интенте «физика» — научные статьи, при «сорт яблока» — садоводство, при «песня» — аудио.

**Задача 3:**

- a) Precision@3 = 2/3 ≈ 0.67; Recall@3 = 2/3 ≈ 0.67.

- б) NDCG@5 = 0.906 (расчёт выше).

**Задача 4:**

- a) 1) «принтер Epson L3250 не работает обновление Windows 11»; 2) «ошибка принтера L3250 после win11 update».

- б) Галлюцинации, потеря намерения, подмена модели/бренда принтера.

- в) Автоматическая метрика: сравнение с релевантными документами по BLEU или ROUGE; human-in-the-loop: проверка экспертами на валидность.
