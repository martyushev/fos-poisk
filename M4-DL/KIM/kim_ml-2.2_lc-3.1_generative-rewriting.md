## Эталон выполнения

**Пример правильного ответа:**

Генеративные модели могут использоваться для query rewriting в следующих сценариях:

1. **Расширение запроса синонимами и связанными терминами:** T5 генерирует альтернативные формулировки запроса для увеличения полноты поиска.

2. **Переформулирование разговорных запросов в поисковые:** BART преобразует естественно-языковой запрос «как починить кран, если он капает?» в «ремонт крана капает причина» для лучшего совпадения с документами.

3. **Коррекция неправильной раскладки клавиатуры:** GPT может исправить запросы, набранные не в той раскладке, на основе контекста.

Риски: галлюцинации (модель генерирует термины, которых нет в документах) — можно смягчить через фильтрацию по словарю; потеря намерения — валидация через классификатор интента; повышенная latency — использование кэширования или более лёгких моделей.
